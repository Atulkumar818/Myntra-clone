# Myntra-clone
A front-end clone of the Myntra e-commerce website built using HTML, CSS, and JavaScript. This project replicates the core UI and basic functionality of an online shopping platform, including product listing and shopping bag management.

Features
 Dynamic product listing using JavaScript
 Add to Bag functionality
 Data persistence using localStorage
 Search bar UI
 Shopping bag item count indicator
 Clean and responsive UI inspired by Myntra

How It Works
Product Data is stored in items.js
JavaScript dynamically renders items on the homepage
When a user clicks "Add to Bag":
Item ID is stored in localStorage
Bag count updates instantly
The bag icon reflects the total number of items added

Technologies Used
HTML5
CSS3 (Flexbox for layout)
JavaScript (DOM manipulation & localStorage)
